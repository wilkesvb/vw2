Generates a list of posts using shortcode, useful for creating a sitemap or archive page.

== Description ==

For search engine spiders is usefull to scan sitemaps of websites. Visitors similarly like to see list of all posts on blogs. Theses lists are much easier to browse and they are great for SEO.


<h3>Usage</h3>

<p>Simple posts list generates list of all post or pages using shortcodes or function call from template. <pre>[posts]</pre> shortcode and you'll have them automagicly.</p>
<p>
<h4>Examples (for posts):</h4>
<h5>Generate post list using shortcode:</h5>
<pre>[posts]</pre>
<h5>Generate post list using function call:</h5>
<pre>&lt;?php posts_list (); ?&gt;</pre>
<h4>Examples (for pages):</h4>
<h5>Generate list of pages using shortcode:</h5>
<pre>[pages]</pre>
<h5>Generate list of pages using function call:</h5>
<pre>&lt;?php pages_list (); ?&gt;</pre>
</p>

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
